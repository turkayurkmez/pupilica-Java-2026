# ATM Simulator

## Öğrenme Hedefleri

Bu proje, Java programlama dilinin temel kavramlarını öğrenmek için tasarlanmış basit bir ATM simülatörüdür.

## Projede Açıklanan Konular

- **Sabit Değişkenler (final)**: `final` anahtar kelimesi ile değişmeyen değerlerin tanımlanması
- **Değişken Tipleri**: 
  - `String` - Metin verileri
  - `double` - Ondalıklı sayılar
  - `int` - Tam sayılar
- **Sayı Formatları**: Büyük sayılarda okunabilirlik için underscore (`_`) kullanımı
- **Matematiksel İşlemler**: Toplama, çıkarma gibi temel aritmetik işlemler
- **Tip Dönüşümü (Type Casting)**: `double` tipinden `int` tipine dönüştürme
- **Konsol Çıktısı**: `System.out.println()` ile ekrana bilgi yazdırma
