# Yaş Hesaplayıcı

## Öğrenme Hedefleri

- Kullanıcıdan klavye girişi almayı öğrenmek
- Değişkenlerle çalışmayı pekiştirmek
- Temel aritmetik işlemleri uygulayabilmek
- LocalDate sınıfını kullanarak güncel tarih bilgisine erişmeyi öğrenmek
- Konsol tabanlı basit bir uygulama geliştirebilmek

## Projede Açıklanan Konular

- **Scanner Kullanımı**: Kullanıcıdan klavye girişi alma
- **Değişken Tanımlama**: `int` veri tipi ile değişken oluşturma
- **LocalDate Sınıfı**: `LocalDate.now().getYear()` ile güncel yıl bilgisini alma
- **Aritmetik İşlemler**: İki sayı arasında çıkarma işlemi
- **Konsol Çıktısı**: `System.out.println()` ile ekrana yazdırma
- **Kod Yorumlama**: Çok satırlı yorum blokları kullanımı
